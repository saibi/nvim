## tree-sitter cli
$ npm install -g tree-sitter-cli

or 

# ubuntu 22.04 에서 libc 버전이 낮아 최신 cli 가 호환안될 수 있음. cargo 로 설치 
# cargo 설치시 clang 에러 발생하면 패키지 추가 설치 sudo apt install libclang-dev
$ cargo install tree-sitter-cli
